SITIO DE MAQUILLAJE!

El maquillaje es la tecnica destinada a realzar los rasgos faciales y simultaneamente disimular los efectos para conseguir mayor armonia.Conjunto de productos cosmeticos que se aplican sobre la piel, especialmente la de la cara, para maquillarla.

PRODUCTOS QUE SE NECESITAN PARA UN BUEN MAKE UP:

Lo principal es..

  1. Hidratar la piel,Aplicar una crema hidratante habitual, Una crema para el contorno de ojos (si la usas)
  2. Corrector en crema: cubre o disimula alguna imperfeccion como un granito o alguna mancha, se aplica con una brocha pequeña sintentica.
  3. Prebase o primer: para que la sombra de ojos dure mas tiempo y no se cuartee.
  4. Corrector liquido:para tapar las ojeras es necesario hacer dos trazos, uno desde el final del ojo hacia abajo y otro desde el lagrimal hacia abajo, procurar que el triangulo no llegue hasta el pomulo, para difuminar se utiliza una brocha plana sintetica o una esponja beauty.

PRODUCTOS DE LIMPIEZA FACIAL:

  • Agua miselar
  • Jabon liquido
  • Exfoleante
  • Tonico de limpieza
  • Crema hidratante o nutritiva
  • Acido ialuronico
